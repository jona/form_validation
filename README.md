#Form validation

**Simple form validation class**

##Initialize
```coffeescript
form = new app.Form()
```

##Call the ```validate``` function

```coffeescript

form.validate(
  [
    {name: "Text", el: $("[name='comment[text]']")},
    {name: "timestamp", el: $"([name='comment[timestamp]']"), pattern: /[0-9]{1,2}:[0-9]{1,2}:[0-9]{1,2}/}
  ]
)
```
